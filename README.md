# MusiGUI

GUI Library for Spigot, Sponge, and related forks.

Groovy:

```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'com.github.musician101.musigui:<module>:<version>'
}
```

Kotlin DSL:

```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    implementation("com.github.musician101.musigui:<module>:<version>")
}
```
