# MusiGUI
GUI Library for Spigot, Sponge, and related forks.

Groovy:
```Groovy
repositories {
    maven {
        url = 'https://jitpack.io'
    }
}

dependencies {
    implementation 'io.musician101.musigui:<module>:<version>'
}
```

Kotlin DSL
```Kotlin
repositories {
    maven("https://jitpack.io")
}

dependencies {
    implementation("io.musician101.musigui:<module>:<version>")
}
```
